# Storage Optimization

Scripts that tune Ubuntu storage behaviour: I/O schedulers, fstab options, write batching, swap, and media-specific optimizations.

> Maintained by [Blackout Secure](https://blackoutsecure.app)

## Structure

```
linux/ubuntu/storage-optimization/
  <target>/
    <action>-<target>.sh
    README.md
```

## Available Targets

| Target | Folder |
|---|---|
| USB boot | [`usb-boot/`](usb-boot/) |
